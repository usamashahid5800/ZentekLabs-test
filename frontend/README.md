# Products Frontend (React + Vite)

## Setup

```powershell
npm.cmd install
```

## Run in development

```powershell
npm.cmd run dev
```

App runs on `http://localhost:5173`.

## API URL

Default API base URL is:

`https://localhost:7236`

You can override with environment variable:

`VITE_API_BASE_URL`

Example:

```powershell
$env:VITE_API_BASE_URL="https://localhost:7236"
npm.cmd run dev
```

## Build

```powershell
npm.cmd run build
```
